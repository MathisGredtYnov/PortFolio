Groupie-Tracker

Voila probablement le projet que j'ai le moins aimé de l'année. Son but était de nous faire travailler avec un API pour nous permettre de récupérer des données d'un API, puis de les afficher dans un site Web.

Ce projet était en solo, et je trouve ça bien dommage même si d'un point de vue apprentissage c'est mieux que chacun éssaie.

Pour ma part je ne suis pas fier du résultat car je n'ai pas eu assez de temp pour réaliser à bien se projet.

Compétence acquise

  • Travailler avec un API
  • Relier le Web et les données d'un API
  • Travailler en solo
